# Campfire Express
Generated by [Rojo](https://github.com/rojo-rbx/rojo) 7.6.1.

Campfire Express is a Roblox survival/tower-defense game inspired by:

- 99 Nights in the Forest
- Dead Rails
- survival/base defense games
- procedural exploration

## Features

- 🔥 Campfire survival system
- ❄️ Temperature & cold mechanics
- 🌙 Day/night cycle
- 👾 Enemy ECS AI
- 🛡️ Turrets
- 🪵 Resource gathering
- 🌱 Renewable trees/seeds
- 🎒 Inventory UI
- ⚙️ Knit + Matter ECS + Fusion UI

## Development
To build the place from scratch, use:

```bash
rojo build -o "campfire-express.rbxlx"
```

Next, open `campfire-express.rbxlx` in Roblox Studio and start the Rojo server:

```bash
rojo serve
```

For more help, check out [the Rojo documentation](https://rojo.space/docs).
